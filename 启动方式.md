启动命令：
#后端
cd truthseeker-api
# 激活虚拟环境 (Windows)
.\venv_new\Scripts\activate
# 安装依赖 (仅在首次运行或更新依赖时，必须使用虚拟环境 pip)
venv_new\Scripts\pip.exe install -r requirements.txt
# 启动服务
python -m uvicorn app.main:app --reload --port 8000

# 前端（端口 3000）
cd truthseeker-web
npm install  # (仅在首次运行或更新依赖时)
npm run dev  # 启动后访问 http://localhost:3000


后期安装好环境后可直接：
后端：
cd truthseeker-api
.\venv_new\Scripts\activate
python -m uvicorn app.main:app --reload --port 8000

前端：
cd truthseeker-web
npm run dev  # 启动后访问 http://localhost:3000

## 环境变量配置

首次运行前，必须配置以下环境变量文件：

### truthseeker-web/.env.local
```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=sb_publishable_...
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```

### truthseeker-api/.env
```env
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_SERVICE_ROLE_KEY=eyJ...
SUPABASE_JWT_SECRET=your-jwt-secret
KIMI_API_KEY=your-kimi-key
REALITY_DEFENDER_API_KEY=your-rd-key
VIRUSTOTAL_API_KEY=your-vt-key
EXA_API_KEY=your-exa-key
```

## 验证步骤

1. 后端启动后访问 http://localhost:8000/docs 查看 Swagger UI
2. 前端启动后访问 http://localhost:3000
