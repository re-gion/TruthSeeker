启动命令：
#后端
cd truthseeker-api
# 激活虚拟环境 (Windows)
.\venv_new\Scripts\activate 
# 安装依赖 (仅在首次运行或更新依赖时)
pip install -r requirements.txt
# 启动服务
python -m uvicorn app.main:app --reload --port 8000

# 前端（端口 3000）
cd truthseeker-web
npm install  # (仅在首次运行或更新依赖时)
npm run dev  # 启动后访问 http://localhost:3000


后期安装好环境后可直接：
后端：
cd truthseeker-api
.\venv_new\Scripts\activate 
python -m uvicorn app.main:app --reload --port 8000

前端
cd truthseeker-web
npm run dev  # 启动后访问 http://localhost:3000
